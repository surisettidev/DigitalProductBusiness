# Marketing Content

This folder is auto-populated daily by the AI agent (see `workflows/ai-agent-instructions.md`).

## Structure
```
marketing/
└── YYYY-MM-DD/
    ├── twitter.md    — 3 X/Twitter posts
    ├── reddit.md     — 1 Reddit comment draft
    ├── linkedin.md   — 1 LinkedIn post
    └── email.md      — 1 nurture email
```

## Email templates
See `email-templates/` for transactional emails (welcome, delivery, purchase confirmation).
